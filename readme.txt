ProjectName: LoyaltySystemsChecker
Description: Приложение для создания отчетов по кнопкам лояльности
Author: nn-admin@bzklinika.ru 31-555

При запуске без параметров выполняется проверка работоспособности кнопок по всем филиалам, указанным в настройках в секции daily_check. В случае если за дня нет данных, то будут высланы заявки на адреса из секции notify_daily_errors.

При запуске с ключом '-week' будет выполнена рассылка отчетов за прошлую неделю:
1) по опросу "порекомендуете ли вы нашу клинику" на адреса главных врачей, указанных в разделе настроек notify_reports
2) по опросу "оцените качество приема" будет сформирован документ в формате XLSX и разослан на адреса, указанные в разделе настроек commercial_department

При запуске с ключом '-month' выполняются действия, аналогичные недельному отчету, но только за прошедший месяц.

Для выполнения скрипта необходимо создать соответствующие задания в планировщике заданий.
